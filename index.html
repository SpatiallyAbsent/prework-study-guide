HTML

  • HTML stands for HyperText Markup Language
  • The head element contains information about the webpage.
  • The head contains metadata.
  • The body element represents the visible content shown to the user.
  • Elements begin at the opening tag, contain content, and end after the closing tag.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS containes Selectors and declarations
  • A declaration contains a property and a value, such as the color:blue when assigning collor
  • Multiple delcarations in a selector are known as declaration blocks
  • The DRY code is in reference to adding multiple elements when styling in CSS and you want them all to share the same properties.
  • External CSS is when CSS changes are kept in a seperate file.
  • Internal CSS are styles that will apply to a single page.
  • Inline CSS is used to style a specific HTML tag.
